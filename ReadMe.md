#### Instructions
After cloning, `cd` into the `yapc` directory and run the command
```bash
cmake -S . -B ./build
```

Then `cd` into the `build` directory and run `make`. Then run the binary with `./demo`

In summary
```bash
git clone "https://github.com/terencendabereye/YAPC.git"
cd yapc
cmake -S . -B ./build
cd ./build
make
./demo
```
